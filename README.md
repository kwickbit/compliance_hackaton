# compliance

If running local, use subquery_sandbox

## start
 
 Set environment variables:
 ```
 cp .env.example .env
 ```
 Let it like this if you are running subquery_sandbox locally. If not choose the URL for your graphql endpoint.
 
 go to docker and then:

```
bash run.sh
```

then 
```
cd my-next-app
```

then
```
yarn dev
```
